# Tailwind Color Gradient Generator

A perceptually accurate color palette generator that creates Tailwind-style color gradients using the OKLCH color space.

## Features

- **OKLCH-Based Generation**: Uses the OKLCH color space for perceptually uniform color gradients
- **Tailwind-Style Palettes**: Generates the full 11-shade palette (50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950)
- **Smart Color Adjustment**: Automatically adjusts hue and chroma to maintain vibrant, natural-looking colors
- **Interactive Interface**: Real-time color picker with hex input
- **Copy to Clipboard**: Easy copying of individual color values

## How It Works

### OKLCH Color Space

Unlike traditional HSL or RGB color spaces, OKLCH provides:

- **Perceptual Uniformity**: Changes in lightness (L) produce consistent visual changes across all hues
- **Predictable Results**: A 10% increase in lightness looks the same whether applied to blue, yellow, or any other color
- **Natural Gradients**: Allows hue and chroma adjustments to create more natural-looking color scales

### Color Generation Algorithm

1. **Base Color (500)**: Your input color is placed at the 500 shade
2. **Lighter Shades (50-400)**: Generated by increasing lightness and reducing chroma proportionally
3. **Darker Shades (600-950)**: Generated by decreasing lightness while adjusting chroma to maintain vibrance
4. **Perceptual Accuracy**: Unlike simple brightness adjustments, the algorithm allows hue to shift naturally for better perceptual results

### Lightness Values

The generator uses carefully calibrated lightness values to match Tailwind's visual progression:

- **50**: L ≈ 0.97 (Very light)
- **100**: L ≈ 0.93
- **200**: L ≈ 0.85
- **300**: L ≈ 0.75
- **400**: L ≈ 0.65
- **500**: Base color lightness (your input)
- **600**: L ≈ 85% of base
- **700**: L ≈ 70% of base
- **800**: L ≈ 55% of base
- **900**: L ≈ 40% of base
- **950**: L ≈ 25% of base (Very dark)

## Usage

1. Open `index.html` in a web browser
2. Choose a color using the color picker or enter a hex code
3. Click "Generate" to create the palette
4. Copy individual colors as needed

## Technical Details

The application includes:

- Full RGB ↔ Oklab ↔ OKLCH conversion utilities
- Proper gamma correction for linear RGB transformations
- Chroma scaling to maintain color vibrance at different lightness levels
- RGB clamping to ensure all generated colors are valid

## Browser Support

Works in all modern browsers that support:
- Native color input type
- ES6 JavaScript
- CSS3

## About OKLCH

OKLCH is a modern color space designed by Björn Ottosson in 2020. As of 2025, it's widely supported in browsers and is the default color system in Tailwind CSS v4.
